# Kream Crawling

크림 웹사이트에서 다음 두 가지 데이터를 수집하는 간단한 크롤링 프로젝트입니다.

## 1️⃣ 검색 결과 수집
`search` 명령은 사용자가 입력한 제품명을 기준으로 크림 검색 페이지에 접속한 뒤, 검색 결과 목록에서 상품 카드 정보를 수집합니다.

```bash
python src/main.py search --keyword "조던 1 블랙 모카" --max-items 30
```

수집되는 주요 정보는 다음과 같습니다.

1. 상품 URL
2. 상품 ID
3. 상품명 추정값
4. 현재 가격
5. 관심 수
6. 리뷰 수
7. 거래 수
8. 검색어

검색 결과는 CSV 파일로 저장되며, 이후 거래 데이터 수집의 입력 파일로 사용할 수 있습니다.

## 2️⃣ 거래 데이터 수집
`transactions` 명령은 상품 URL이 들어 있는 CSV 파일을 입력으로 받아, 각 상품 상세 페이지의 거래 데이터를 수집합니다.

```bash
python src/main.py transactions --input data/output/search_20260409_144424_조던1블랙모카.csv --tab "체결 거래"
```

수집되는 주요 정보는 다음과 같습니다.

1. 상품 URL
2. 탭명
3. 모델번호
4. 발매일
5. 옵션
6. 가격
7. 거래 시점
8. 수집 시각
